Welcome to Fontastic! 

This program has been designed to make the font integration process easier and more accurate at Wunderkind. Fontastic will pre-fill many inputs for you, finds and separates useful information, and offers suggestions to help along the way. For more detailed information on the font integration process, please refer to the VI-UXE Confluence content:

https://wunderkindco.atlassian.net/wiki/spaces/CE/pages/379912353/Font+Integrations

*** String.prototype.includes ***

Source: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/includes

This method was included to make it easier to search for substrings within larger strings. This is used in many parts of Fontastic to verify that it has found the requested piece of information.
